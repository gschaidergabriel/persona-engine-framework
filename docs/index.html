Persona Engine Framework

What this demo shows

This is an honest architecture demo of the Persona Engine Framework. It does not simulate live AI responses or fake API calls.

The core idea is simple: persona is not a prompt. A persona is a persistent constraint system (tone, structure, priorities) that lives outside the interaction loop.

Because browsers cannot reliably control third‑party AI sessions, this demo shows the workflow explicitly instead of pretending automation.

You copy a persona‑anchored system prompt, open ChatGPT, and continue the conversation there — transparently and reproducibly.
